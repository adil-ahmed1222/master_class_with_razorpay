## 10. Live Workflow Demo (on-page “Watch Demo” pipeline)

The **Live Workflow** section is an interactive scroll demo of an **AI employee pipeline** — the same architecture attendees wire in the masterclass.

**Section title:** Every Node in Your AI Employee  
**Subtitle:** Scroll the orbit to open each step—from prompt in to response out. This is the same pipeline you will wire live in the masterclass.

### Six pipeline nodes

| Node | Role |
|------|------|
| **Input** | Receives user prompts and normalizes context for downstream agent nodes. |
| **Planner** | Breaks goals into steps, selects tools, and routes work to specialized nodes. |
| **Retriever** | Fetches relevant documents and embeddings from the knowledge store. |
| **Tools** | Executes MCP and API tools; returns structured results to the planner. |
| **Memory** | Persists session state, long-term facts, and vector indexes for recall. |
| **Output** | Synthesizes final responses with citations and streams tokens to the client. |

This is the **“demo”** on the website — not a separate product, but a visual explanation of how an agent processes work from input to output.
